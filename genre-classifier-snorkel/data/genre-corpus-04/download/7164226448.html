Note:  In order to view the Quicken Deluxe User Guides, you must have the free Adobe Acrobat Reader installed on your computer.

 

To download the Quicken for Windows Guides:

  1. Click the link below to start the download for your version of Quicken.
  2. At the File Download dialog box, choose Save This Program to Disk, and then click OK.
  3. At the Save As window, select a destination to download the file, and then click Save. It is important to make note of the file name at this point.
  4. Once the download is complete, double-click the file to decompress. It is important to note the location of the file.

To download the Quicken for Macintosh Guides:

  1. Hold down the Control key and click the link below for the Installation or User guide for your version of Quicken.
  2. Choose Download Link to Disk from the menu.
  3. In the Save window, select a destination to download the file, and then click Save.
  4. Once the download is complete, double-click the file to open it.
