Downloading Mutt

The current stable public release version is 1.4.1.

CVS

CVS is available from cvs.mutt.org in the /home/roessler/cvs directory:

cvs -d :pserver:anonymous@cvs.mutt.org:/home/roessler/cvs login
cvs -z3 -d :pserver:anonymous@cvs.mutt.org:/home/roessler/cvs co mutt

The password is anonymous.
