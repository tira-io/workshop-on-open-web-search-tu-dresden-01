Boot problem with boot floppy

Hello!
I formatted the C partition of HDD and transfered system
files from Win98SE bootdisk. When I restart the computer,
it says "no system media, boot failure". I want to boot
from HDD and then switch to my WinME installation folder in
D drive to install manually. What is the problem? Any ideas??

Re: Boot problem with boot floppy

Reboot using the Windows 98SE bootdisk and format the hard disk again, this
time using the following switch:

format c: /s

This should copy over the system files required to boot your machine from
the hard disk.

Rich

Re: Boot problem with boot floppy

Check that the partition is set as active (bootable) by booting from your
floppy and running FDISK. Also ensure that BIOS is set to boot from your
hard disk.
