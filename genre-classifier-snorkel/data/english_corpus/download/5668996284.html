If you just want to get started with a full-featured weblog, we recommend TypePad, our simple weblog service, which is based on the Movable Type engine and requires no installation. TypePad weblogs are easy to customize and offer features like photo albums without requiring you to have a web server or any technical knowledge.

Note About Licensing: Movable Type is free for personal or non-profit use, though donations are greatly appreciated. Movable Type is not free for use on commercial sites. See the commercial download page for information on commercial licensing.

Full Version, with Libraries: this archive contains the full Movable Type system and all of the external libraries needed by the system. If you're not sure whether your hosting provider has the external modules installed, select this version, and follow the installation instructions.

Full Version: this archive contains the full Movable Type system, but none of the external libraries needed. If you know that your hosting provider has all of the modules you'll need installed already, select this version, and after downloading the archive follow the installation instructions.

Upgrade Version: this archive contains the pieces of the Movable Type system that you will need to upgrade from a previously-installed version of the software. If you are already running Movable Type, select this version, and after downloading the archive follow the upgrade instructions.
