The existing SSH Secure Shell products have been re-branded with new product names starting from the new release version 4.0. The new product names for SSH Secure Shell for Workstations, Servers, and Windows Servers are now:


  • SSH TectiaTM Client (previously SSH Secure Shell for Workstations)
  • SSH TectiaTM Server (Unix) (previously SSH Secure Shell for Servers)
  • SSH TectiaTM Server (Windows) (previously SSH Secure Shell for Windows Servers)
As a part of the SSH Tectia solution, we have also announced a new transparent desktop security product called SSH TectiaTM Connector, which is based on the original Secure Shell technology.

Due to the fact that the new SSH Tectia solution has been developed to best suit the needs of large enterprises, governmental organizations and financial institutions, non-commercial versions of SSH Tectia solution is not available.

SSH Tectia client / server solution will be available for non-commercial use with discounts on list prices. For more information on new products and discounts for non-commercial use, please contact SSH sales representative for any sales related questions or submit an inquiry at http://www.ssh.com/company/sales/inquiry.mpl.

You can still download the SSH Secure Shell 3.2 non-commercial source code and Windows Client free of charge from various anonymous ftp sites around the globe for purposes of EVALUATION, NON-COMMERCIAL USE, AND UNIVERSITY USE as defined in the license agreement.

Please note that the non-commercial version has limited functionality. For an evaluation of full functionality, please download the SSH Tectia client / server solution commercial evaluation version.
