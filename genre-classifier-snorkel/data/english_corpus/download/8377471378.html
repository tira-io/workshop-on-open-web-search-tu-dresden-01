Release build -- 1.5.10 (stable)

This is the current release of VirtualDub for end users, containing the executable and supporting files. Unzip the contents into a new folder and run VirtualDub.exe. (Do not run it directly from the zip file!) There is no version of 1.5.10 specifically optimized for P4 (although the general build does have some P4 optimizations).

List download mirrors for V1.5.10 release build (VirtualDub-1.5.10.zip)

Source code

You'll want this file to look at the innards of VirtualDub and to recompile the program. This codebase will build both the normal and P4 versions. All tools necessary to build the release executable and help file are included (verinc, mapconv, disasm, and Lina). You need Microsoft Visual C++ 6.0 Service Pack 5 + Processor Pack or later to build the general release, and Intel C/C++ 6.0 or later to build the P4 version.

Note: The source code archive is an uncompressed zip file packed using the bzip2 block-sorting compressor.
