Fresh install of windows, without ANYTHING else? So no drivers, updates, anything? And you formatted before the new install? Have you added anything new recently (drivers/hardware)? I was having a similar problem. CMOS resets worked but then it would only load Windows with the BIOS failsafe defaults set. After that I could reboot and put the BIOS back how it should be. Turned out my CMOS battery was going flat. Replaced it and its been smooth sailing ever since.
